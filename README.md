ExamGenie
Introduction
ExamGenie is a simple, automated tool that turns your study material into practice exams. Simply provide the scope of your study, and let it generate a tailored quiz to help you prepare for your next test. No more manual note-taking—just study!

Key Features
Smart Quiz Generation: Instantly creates questions based on the text or topics you provide.

Customizable Difficulty: Adjust the complexity of the questions to suit your study needs.

Lightweight & Fast: Built to be run locally, so you can test yourself anywhere, anytime.

How to use
Setup: Clone this repository to your local machine:

Bash
git clone https://github.com/yourusername/examgenie.git
cd examgenie
2. **Run:** Input your study notes and run the generator:
   ```bash
   python generate_exam.py --input "your_notes.txt" --difficulty "medium"
